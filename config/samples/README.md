# Grid Custom Resource Samples

This directory contains sample Grid custom resources for development and
documentation. They are examples only; they do not install the operator or a
Praxis AI gateway.

For the installable Grid package, see [deploy/README.md](../../deploy/README.md).

For a fuller single-cluster walkthrough, see
[deploy/examples/single-cluster-api-provider](../../deploy/examples/single-cluster-api-provider/).
